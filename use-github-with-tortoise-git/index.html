Гитхаб с Черепахой

В последнее время Гит, в том числе благодаря Гитхабу, набирает популярность среди пользователей SCM. От чего же не вспомнить про старую добрую Черепашку, знакомую многим по TortoiseCVS и TortoiseSVN? Эта статья может быть полезна тем, кто хочет делиться кодом через Гитхаб, но не готов ради этого променять правый клик в проводнике на консоль Windows. Осторожно - много картинок.

Установка и настройка

Вам понадобится
  • TortoiseGit – готорит за себя
  • MsysGit – Git для Windows, нужен TortoiseGit (берём файл Git-*.exe)
  • PuTTY – используется для аутентификации на Гитхабе по ключу SSH (берём файл putty-*-installer.exe)
Установка TortoiseGit

  1. Установите PuTTY, который будет управлять ключами SSH (Вам не потребуется не SSH клиент PuTTY, а PuTTYgen и Pagent).

  2. Запустите установщик msysgit, он понадобится для TortoiseGit

    1. Выберите "Git Bash here"

    2. Выберите Use Git Bash only

    3. Выберите Checkout Windows style line endings

  3. Установите TortoiseGit

    1. Если спросит путь к msysgit, укажите директорию bin в установочной директории msysgit (например C:\Program Files\Git\bin)
    2. Установив TortoiseGit, зайдите в TortoiseGit > Settings > Git > Config и проверяем что галочки AutoCRLF и SafeCRLF установлены

      Здесь же можно указать и свё имя и Email, которые будут видеть другие пользователи, читая Ваши коммиты.
